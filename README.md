## 技术栈与环境
- **RTOS 版本**：FreeRTOS Kernel V10.4.6
- **硬件平台**：正点原子 STM32F407 开发板（兼容其他 STM32F4 系列）
- **开发工具**：Keil MDK5（ARMCC 编译器）
- **核心依赖**：STM32F4xx 标准外设库 / HAL 库
- **调试工具**：串口助手（波特率 115200，8N1）、J-Link / ST-Link 下载器

## 核心内容
### 1. 学习笔记（Note）
笔记基于B站正点原子FreeRTOS教程整理，包含**原理讲解 + API 解析 + 实验**，提供两种格式：
- **PDF 版**：简洁排版，适合打印与快速查阅，覆盖所有核心知识点。
- **HTML 版**：可交互阅读，支持目录跳转、代码高亮，便于在线浏览与检索。

### 2. 实战项目（FreeRTOSProjects）
项目按 FreeRTOS 核心模块递进式设计，从基础移植到高级特性，逐步深入：
| 项目序号 | 项目名称 | 核心目标 |
|----------|----------|----------|
| 01 | FreeRTOS 移植 | 将 FreeRTOS v10.4.6 内核移植到 STM32F407，完成最小系统启动 |
| 02-03 | 任务创建与删除 | 掌握动态/静态任务创建、删除与内存管理 |
| 04 | 任务挂起和恢复 | 实现任务状态切换与优先级动态调整 |
| 05 | 中断管理 | 配置中断优先级，在中断中安全调用 FreeRTOS API |
| 06 | 列表与列表项 | 理解 FreeRTOS 内核链表数据结构，实现插入/删除操作 |
| 07 | 开启任务调度器 | 启动调度器，观察多任务并发执行流程 |
| 08 | 时间片调度 | 验证同优先级任务时间片轮转调度机制 |
| 09 | 任务状态查询 | 调用 API 获取任务状态、优先级与栈使用情况 |
| 10 | 任务运行时间统计 | 统计各任务 CPU 占用率，分析系统负载 |
| 11 | 时间延时 | 对比相对延时与绝对延时的差异与适用场景 |
| 12 | 队列操作 | 实现任务间数据传递与同步，验证队列阻塞机制 |
| 13 | 二值信号量 | 用二值信号量实现任务与中断的同步 |
| 14 | 计数型信号量 | 用计数型信号量实现资源计数与限流 |
| 15 | 优先级翻转 | 复现经典优先级翻转问题，分析成因 |
| 16 | 互斥信号量 | 用互斥信号量解决优先级翻转，理解优先级继承机制 |
| 17 | 队列集操作 | 监听多个队列，实现多源数据分发 |
| 18 | 事件标志组 | 用事件标志组实现多事件的同步与等待 |
| 19-22 | 任务通知 | 用任务通知模拟二值信号量、计数型信号量、消息邮箱与事件标志组，体现轻量级同步优势 |
| 23 | 软件定时器 | 实现周期/单次软件定时器，验证回调函数执行 |
| 24 | 低功耗 Tickless 模式 | 配置 Tickless 模式，降低系统功耗 |
| 25 | 内存管理 | 基于 heap_4.c 实现动态内存分配与释放，分析内存碎片 |

## 快速上手
### 1. 克隆仓库
```bash
git clone https://github.com/你的用户名/FreeRTOS-Practice-Notes.git
```

### 2. 运行实验
1. 进入 `FreeRTOSProjects/` 目录，选择目标实验文件夹（如 `02-任务创建和删除-动态/`）。
2. 用 Keil MDK5 打开工程文件（`*.uvprojx`）。
3. 连接 ST-Link/J-Link 下载器，编译工程并下载到 STM32F407 开发板。
4. 打开串口助手，配置波特率 115200，查看实验运行日志。

### 3. 查阅笔记
- 打开 `Note/FreeRTOS学习笔记.pdf` 或 `Note/FreeRTOS学习笔记.html`，对照实验查看原理与 API 说明。
